Custom Roles for Access Manager

== Description ==

Map WordPress roles to KB Roles (KB Groups).

== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-custom-roles.zip.
2. Unzip the file echo-custom-roles.zip.
3. Upload the 'echo-custom-roles' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=access-manager
