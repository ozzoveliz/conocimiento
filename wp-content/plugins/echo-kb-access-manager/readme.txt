Access Manager enables companies and organizations to control and restrict access to their Knowledge Base.

== Description ==

Access Manager enables administrators, companies and organizations to control access to their Knowledge Base(s).

Administrators and KB Managers (a KB specific role) can restrict access to KB Articles, KB Categories, and KB administrative pages.


== Frequently Asked Questions ==

= Where can I find complete documentation on your knowledgebase? =
Full searchable docs can be found at [Access Manager Documentation](https://www.echoknowledgebase.com/documentation/?top-category=access-manager "plugin documentation")

= What browsers work with Echo Knowledge Base? =
We support the current major release and one past major release of the following browsers: Internet Explorer/Edge, Firefox, Safari, and Chrome.
