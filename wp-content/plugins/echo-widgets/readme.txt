This Widget extension lets you place KB Widgets, Shortcodes and Sidebars anywhere on your website.

== Description ==

Add KB Search, Most Recent Articles and other Widgets and Shortcodes to your articles, sidebars and pages.

== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-widgets.zip.
2. Unzip the file echo-widgets.zip.
3. Upload the 'echo-widgets' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=add-ons
