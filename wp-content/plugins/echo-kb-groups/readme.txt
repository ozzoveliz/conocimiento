KB Groups And KB Roles for Access Manager

== Description ==

Create KB Groups and KB Roles to control access to Knowledge Base content.

== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-groups-and-users.zip.
2. Unzip the file echo-groups-and-users.zip.
3. Upload the 'echo-kb-groups' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=access-manager
