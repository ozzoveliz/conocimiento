== Description ==

Elegant Layouts adds Grid and Sidebar Layouts. Use Grid Layout or Sidebar Layout for KB Main page or combine Basic, Tabs, Grid and Sidebar layouts in a variety ways.

== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-elegant-layouts.zip.
2. Unzip the file echo-elegant-layouts.zip.
3. Upload the 'echo-elegant-layouts' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=add-ons
