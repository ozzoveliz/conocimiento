Ajoutez une disposition en grille et des barres latérale à la page principale ou faites des combinaisons avec les mises en forme de votre Base de connaissances de façon cool.
Ceci est un complément de l'extension echo-knowledge-base.

== Description ==

Elegant Layouts ajoute des mises en formes pour la disposition en grille et pour les barres latérales. Utilisez la disposition en grille avec ou sans barre latérale pour la page principale de la Base de connaissances ou faites des combinaisons avec les mises en forme classiques, tabulation, grille et colonne latérale de façon cool.

== Installation ==

= Mise à jour de l'extension manuellement =

1. Téléchargez le fichier 'echo-elegant-layouts' à partir de la page de l'extension.
2. Décompressez localement le fichier echo-elegant-layouts.zip.
3. Envoyez le dossier 'echo-elegant-layouts' vers le répertoire '/wp-content/plugins' de votre site (ne pas renommer le fichier).

Vous pouvez aussi lire les recommandations sur https://www.echoknowledgebase.com/documentation/?top-category=add-ons
